# cancer-multiomics-network-pharmacology
Multi-omics and network pharmacology analysis of anticancer phytochemicals in neuroblastoma models
# Multi-Omics Analysis: Cancer Biology Applications

Computational analyses conducted during a research fellowship at AMDI, 
Universiti Sains Malaysia, investigating anticancer mechanisms of 
natural compounds in neuroblastoma (SH-SY5Y) cells.

## Approaches Used
- Metabolomics: MetaboAnalyst, SIMCA (multivariate analysis)
- Network pharmacology: target identification and pathway mapping
- Molecular docking: AutoDockTools, Schrödinger Maestro
- ADMET profiling: SwissADME
- Systems biology: protein-protein interaction networks (STRING, Cytoscape)

## Associated Publication
